airbnb is a full-stack web application inspired by Airbnb where users can explore, create, and manage travel listings. It allows users to browse destinations, view details with maps, and perform CRUD operations securely.
Features
🏡 Create, Edit & Delete Listings
🗺️ Interactive Maps (Mapbox)
📍 Location-based listing display
👤 User Authentication (Login / Signin)
🔐 Secure sessions & password validation
🖼️ Image upload support

=>  Tech Stack

Frontend:
    EJS (Embedded JavaScript Templates)
    HTML, CSS, Bootstrap
Backend:
    Node.js
    Express.js
Database:
    MongoDB Atlas

Other Tools:
    Mongoose
    Cloudinary (for image uploads)
    Map API (TomTom / Mapbox)
    Render (Deployment)
